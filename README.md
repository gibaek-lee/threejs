## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# deploy step
$ npm run generate # generate static project
$ npm run deploy # deploy to githubpages
```
